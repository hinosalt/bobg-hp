# Definition of Done

以下をすべて満たしたら Done:

- [x] `scripts/check.sh` が成功する
- [x] 不変性を検証するスクリプトが追加されている
- [x] README が起動/検証手順を反映している
- [x] `docs/decisions.md` に重要判断が記録されている
- [x] `progress.md` が `Status: DONE` / `DoD: SATISFIED` / `Checks: PASS` になっている

## 注意
- チェック未通過で Done にしない。
- 速度より再現性を優先する。
- DoD は以下3条件を同時充足すること: `Status: DONE` / `DoD: SATISFIED` / `Checks: PASS`。
